🌿 木森玄機 — 八字命理與流年詳批

以現代語言解構古老命理，讓八字不再神祕難懂。

木森玄機 是一套基於「日主強弱法」與「十神流年演算法」所設計的命理分析系統，
透過程式化邏輯與古典命理結構結合，提供：

八字排盤

五行強弱分析

喜用神判定

流年十神預測

事業、財運、感情多面向解析

本專案以「易學程式化」為目標，
讓命理資訊能夠更直覺、更透明、更科學地呈現。

🔍 核心功能

自動排盤： 依西元生日與出生時間生成四柱八字

五行統計圖： 清楚呈現金木水火土強弱

喜用神分析： 根據格局、旺衰、十神生剋判定

流年詳批： 以十神流年推算年度走勢

命格解析： 提供性格、機遇、挑戰等多面向分析

🧠 技術特點

完整以 JavaScript 實作

支援 客製化命理規則

排盤、計算流程 透明化、非黑箱

支援 前端展示（GitHub Pages 可瀏覽）

📜 聲明

本命理系統屬於自然哲學與文化研究性質，
所有解析僅供參考，
人生方向仍掌握於自身選擇與努力之中。
